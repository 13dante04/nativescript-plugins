## Installing 

```base
    ns plugin add @triniwiz/nativescript-stripe
```

## Android

::: warning Note
The minimum required SDK is version 21 (Lollipop :lollipop:)
:::

## iOS

::: warning Note
The minimum required platform version is 11
:::
